## instalación linux
### 1. Distribucion basadas en Debian(Ubuntu):
* Abrir la terminal
    ```
    sudo apt update
    sudo apt install maven
    ```
* Verificar Instalación
    ```
    mvn -version
    ```
### 2. Distribuciones basadas en Red Hat (CentOS, Fedora):
* Abrir la terminal
  * Para Fedora
  ```
  sudo dnf install maven
  ```
  * Para CentOS
  ```
  sudo dnf install maven
  ```
* Verificar Instalación
  ```
  mvn -version
  ```
